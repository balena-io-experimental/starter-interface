## Requisites

- Node ^18
- Yarn Classic

## Install the dependencies

```bash
yarn
```

### Development modes

#### SPA (Single Page Application)

```bash
yarn dev
```

#### Electron

```bash
yarn dev-electron
```

#### PWA (Progressive Web App)

```bash
yarn dev-pwa
```

### Build the app for production

#### SPA (Single Page Application)

```bash
yarn build
```

#### Electron

```bash
yarn build-electron
```

#### PWA (Progressive Web App)

```bash
yarn build-pwa
```

### Linting and formatting

#### Lint the code

```bash
yarn lint
```

#### User [Prettier](https://prettier.io) to format the code

```bash
yarn format
```
